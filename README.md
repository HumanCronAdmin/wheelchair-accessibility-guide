# Wheelchair Accessibility Guide Japan

Search 25+ Tokyo locations with detailed wheelchair accessibility information. Stations, tourist spots, shopping areas, and hotels — all from official sources.

**[Live Site](https://humancronadmin.github.io/wheelchair-accessibility-guide/)**

## Features
- Filter by category (Stations / Tourist Spots / Shopping / Hotels)
- 3-level accessibility rating (Full / Partial / Limited)
- Direct links to official venue websites
- Mobile-first responsive design

## Data Sources
- JR East barrier-free information
- Tokyo Metro accessibility pages
- GO TOKYO official guide
- Individual venue official websites
